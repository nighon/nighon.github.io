# nighon.github.io

个人技术博客 https://nighon.me

## 技术栈

- [GitHub Pages](https://pages.github.com/) 托管服务
- [Jekyll](https://jekyllrb.com/) 静态站点生成器 v4.2.0
- [Markdown](https://daringfireball.net/projects/markdown/) 语法，把纯文本转换成 HTML
- [Liquid](https://github.com/Shopify/liquid/wiki) 模板引擎

## 发布方法

修改 _posts 目录下的文件，提交到 GitHub 后，会自动生成页面。
